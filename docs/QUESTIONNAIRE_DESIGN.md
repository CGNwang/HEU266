# O-Match 問卷模組設計文檔

## 概述
基於前端實現，已完整設計 5 個問卷模組，共 33 道題目，對應用户建檔→生活習慣→性格分析→三觀評估→感情需求的完整人設圖譜。

---

## Module 1: 基礎畫像 (5 題)
**目的**: 建立用户基本信息與匹配物理坐標  
**提示語**: 欢迎开启你的灵魂盲盒。这几个基础问题将作为我们为你匹配的『物理坐标』，仅需 30 秒即可完成。

### 題目列表
| ID | 題目 | 題型 | 選項 |
|----|-----|------|------|
| q1_1 | 你的性别是？ | single | 男生 / 女生 |
| q1_2 | 期待相遇的灵魂是？ | single | 男生 / 女生 / 都可以，灵魂契合最重要 |
| q1_3 | 你所在的学段是？ | single | 大一大二 / 大三大四 / 硕士 / 博士 |
| q1_4 | 期待伴侣的学段？ | multiple | 大一大二 / 大三大四 / 硕士 / 博士 / 都可以 |
| q1_5 | 你常在哪些地方散步/活动？ | multiple | 14 个校园地点（启航中心、图书馆、各楼宇、体育场等） |

## Module 2: 生活颗粒度 (10 題)
**目的**: 评估日常生活兼容度，避免『水火不容』的匹配  
**提示語**: 灵魂再契合，也需要生活在同一个物理时空。告诉我们你的生活节奏，我们会帮你避开那些『水火不容』的雷区。

### 題目結構
每個大問題包含 1 個主問題 + 1 個子問題：

#### 1. 作息結界 (Q2_1)
- **主問題**: 你的日常生物钟是怎样的？
  - 早起自律派 🌅
  - 弹性凡人派 ⚖️
  - 深夜灵感派 🦉
- **子問題**: 对于对方的作息，你的态度是？
  - A. 必须和我同频 💣
  - B. 最好相似，但我能包容 💛
  - C. 无所谓，互不打扰就行 ⭕

#### 2. 空間信仰 (Q2_2)
- **主問題**: 对于个人空间的整洁度？
  - 极度整洁 ✨
  - 乱中有序 📦
  - 随性洒脱 🌪️
- **子問題**: 如果对方的卫生习惯和你差异极大，你能接受吗？
  - A. 绝对不能接受 💣
  - B. 稍微有点介意 💛
  - C. 完全不介意 ⭕

#### 3. 消息頻率 (Q2_3)
- **主問題**: 你期望日常的微信沟通节奏是？
  - 低频深度型 📝
  - 平衡型 ⚖️
  - 常聊陪伴型 📱
- **子問題**: 如果对方的沟通频率和你不匹配，你会？
  - A. 无法接受，很容易吵架 💣
  - B. 容易失落，但能尝试适应 💛
  - C. 尊重对方节奏 ⭕

#### 4. 煙酒結界 (Q2_4)
- **主問題**: 你对烟/酒的态度是？
  - 烟酒都拒绝 🚫
  - 社交时可以 🍺
  - 无所谓 😎
- **子問題**: 这是你无法协商的底线吗？
  - A. 是的，必须同意 💣
  - B. 有点介意，但可商量 💛
  - C. 完全不是底线 ⭕

#### 5. 社交頻率 (Q2_5)
- **主問題**: 你期望的群聚频率？
  - 宅家小聚 🏠
  - 平衡型 ⚖️
  - 热爱社交 🎉
- **子問題**: 如果伴侣的社交期望和你截然不同，你会？
  - A. 强烈冲突 💣
  - B. 能部分妥协 💛
  - C. 完全尊重彼此 ⭕

## Module 4: 三觀與曠野 (6 題)
**目的**: 评估价值观、人生目标、人生哲学的匹配度  
**提示語**: 三观没有标准答案，但频率相近的人，总能在同一片旷野里看见相同的风景。

### 題目設計

#### Q4_1: 金錢觀
**題目**: 你突然获得一笔 5000 元的"意外之财"，你更倾向于如何使用？
- 🏦 理性储蓄派（长期主义）- 大部分存起来或用于未来
- ⚖️ 平衡分配派（规划享受）- 一部分存起来，一部分奖励自己
- 🎉 即时享乐派（体验优先）- 直接用来提升当下幸福感

#### Q4_2: 生涯規劃型
**題目**: 面对毕业后的去向选择，你更接近哪种想法？
- 🧭 清晰路径型（目标导向）- 明确规划，持续准备
- 🌊 顺势而为型（弹性发展）- 大致方向，灵活调整
- 🧪 探索试错型（开放尝试）- 多尝试不同可能

#### Q4_3: 關係優先級
**題目**: 当学业/工作压力很大，同时另一半希望你多花时间陪伴时，你更可能？
- 🎯 任务优先型 - 先做眼前最重要的事
- ⚖️ 尝试平衡型 - 协调时间，双方都妥协
- ❤️ 关系优先型 - 优先保证陪伴和情感连接

#### Q4_4: 風險偏好
**題目**: 如果有一个机会去一个陌生城市发展 vs 留在熟悉环境稳定发展
- 🏠 稳定优先 - 看重安全感和确定性
- ⚖️ 权衡决策 - 综合考虑收益与风险
- 🚀 冒险驱动 - 抓住改变人生的机会

#### Q4_5: 關係付出觀
**題目**: 在确定关系前的日常相处中，你更认可哪种方式？
- 🧾 边界清晰型 - 倾向 AA，清晰是稳定基础
- ⚖️ 弹性分担型 - 大体均衡，谁方便谁多付
- 🎁 情感驱动型 - 不计算分配，看重付出意愿

#### Q4_6: 生活方式
**題目**: 一个完全自由的周末，你更理想的状态是？
- 📈 自我提升型 - 学习、健身、阅读
- ⚖️ 平衡生活型 - 一部分放松，一部分投入
- 🛋️ 纯放松型 - 彻底休息娱乐

## Module 5: 親密關係說明書 (7 題)
**目的**: 理解愛與被愛的語言、依戀風格、關係期望  
**提示語**: 每个人都是一本待翻阅的书，依恋是序章，爱是语言，冲突是章节里的起伏。

### 題目設計

#### Q5_1: 依戀風格
**題目**: 在亲密关系中，当你感到压力或与对方产生矛盾时，你通常会？
- 🤝 安全型 - 主动沟通，坦诚表达，相信共同解决
- 📡 焦虑型 - 不安反复试探，渴望立刻确认和安慰
- 🏔️ 回避型 - 需要独处冷静，暂时躲避讨论

#### Q5_2: 愛的語言 (多選)
**題目**: 当你感到被爱或被在乎时，以下哪种场景最让你心动？
- 🗣️ 肯定的言语 - 夸赞、真诚文字表达
- 🧑‍🤝‍🧑 精心的时刻 - 专心陪伴、散步、聊天
- 🎁 接受礼物 - 记得喜好、特殊日子准备礼物
- 🫂 服务的行动 - 分担琐事、实际帮助
- 🤗 身体的接触 - 自然肢体接触、拥抱

#### Q5_3: 個人空間觀
**題目**: 在一段亲密关系中，你觉得"个人空间"对你来说意味着什么？
- 🔒 重要边界 - 需要明确独处时间和私人空间
- 🔗 共享融合 - 喜欢两个人一起做，独处容易孤独
- ⚖️ 动态平衡 - 状态好时独立，低落时依赖

#### Q5_4: 情緒支持偏好
**題目**: 当你感到情绪低落或疲惫时，你更希望伴侣怎么做？
- 🫂 陪伴倾听 - 安静陪伴，听倾诉
- 💡 理性分析 - 梳理问题，给出建议
- 🎉 转移注意力 - 出去走走、吃东西、看轻松内容
- 🚪 尊重独处 - 给空间自己消化

#### Q5_5: 安全感來源
**題目**: 你觉得一段关系中，"安全感"主要来源于什么？
- ✅ 确定性 - 言行一致、承诺兑现、消息及时回复
- 🔗 包容度 - 即使犯错也被接纳，不轻易离开
- 🌐 社交融入 - 被介绍给朋友家人
- 🛡️ 边界清晰 - 明确拒绝暧昧，避免猜疑

#### Q5_6: 被消耗的原因
**題目**: 在关系中，你更容易因为什么而感到"被消耗"？
- 🗣️ 无效沟通 - 反复争吵同样问题，回避讨论
- 📉 情绪过载 - 不断承接对方负面情绪被拖垮
- ⚖️ 付出失衡 - 总是自己主动维系，对方回应冷淡
- 🎭 自我压缩 - 为迁就对方压抑自己需求

#### Q5_7: 核心感受期望 (多選)
**題目**: 你更希望亲密关系带给你的核心感受是？
- 🏠 归属感 - 随时可以回去的港湾
- 🚀 成长感 - 互相激励，共同探索更大世界
- 🎭 松弛感 - 完全做自己，不用伪装
- 🔥 激情感 - 保持心动、新鲜感和浪漫

## 設計亮點

1. **漸進式深入**: 從基礎信息 → 生活習慣 → 性格 → 三觀 → 心理需求，逐步揭示人設
2. **雙層設計**: Module 2 采用主問題+子問題設計，評估既主要維度又評估容納度
3. **多維匹配**: 不同 Module 側重不同維度，提升匹配精准度
4. **用戶友好**: 題目多使用 Emoji、隱喻、故事場景，提升填寫體驗
5. **完整性**: 33 題覆蓋基本信息、生活兼容、性格、價值觀、感情需求五大維度
